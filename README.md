# Healthcare Analytics Dashboard

A Big Data Analytics project for cardiovascular disease prediction using Apache Spark, Machine Learning, and Streamlit.

## Features
- Data preprocessing using Apache Spark
- Exploratory Data Analysis
- Spark SQL analytical queries
- Logistic Regression model training
- Real-time disease prediction dashboard

## Tech Stack
- Apache Spark
- Python
- Scikit-learn
- Streamlit
- Plotly

## Model Performance
- Spark Model Accuracy: 78.33%
- Deployment Model Accuracy: 72.81%

streamlit run app.py
