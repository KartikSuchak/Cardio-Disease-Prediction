# ❤️ Cardio Disease Prediction Dashboard

A **Big Data Analytics Project** for predicting cardiovascular disease risk using **Apache Spark**, **Machine Learning**, and **Streamlit Dashboard UI**.

---

## 📌 Project Overview

This project analyzes patient healthcare records and predicts the probability of cardiovascular disease using machine learning models trained on healthcare data.

The project integrates:

- **Apache Spark** for big data preprocessing and analytics  
- **Machine Learning** for prediction  
- **Streamlit** for an interactive healthcare dashboard  

---

## 🚀 Features

- Data Cleaning and Preprocessing using Apache Spark
- Exploratory Data Analysis (EDA)
- Spark SQL Analytical Queries
- Logistic Regression Disease Prediction Model
- Real-Time Interactive Dashboard UI
- Probability-Based Risk Prediction
- Visual Risk Gauge Meter

---

## 🛠️ Tech Stack

- Python  
- Apache Spark / PySpark

- Plotly
